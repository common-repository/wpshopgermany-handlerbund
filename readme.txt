== Description ==

Mit Hilfe dieses Plugins ist es möglich deutsche rechtssichere Texte (für Shops) zu aktualisieren. Die Texte werden vom [Händlerbund](https://www.haendlerbund.de/de/?utm_medium=affiliate&utm_campaign=partnerid_131&utm_source=affiliate.haendlerbund.de "Händlerbund – Konfiguration für den wpShopGermany-händlerbung Wordpress Plugin") bezogen (Abo vorausgesetzt).

Das Plugin wurde für das Wordpress Plugin [wpShopGermany](http://wpshopgermany.de "wpShopGermany – Das erste deutsche Wordpress Shop Plugin") entwickelt. Es kann auch ohne das Shop Plugin von [maennchen1.de](http://maennchen1.de "Maennchen1.de – Entwickler des wpShopGermany Wordpress Shop Plugins") genutzt werden.

Um diesen Dienst nutzen zu können ist ein Account beim [Händlerbund](https://www.haendlerbund.de/de/?utm_medium=affiliate&utm_campaign=partnerid_131&utm_source=affiliate.haendlerbund.de "Händlerbund – Konfiguration für den wpShopGermany-händlerbung Wordpress Plugin") nötig. 

== Installation ==

Zur Installation beachten Sie bitte folgende Schritte:

1. Die Plugindateien nach wp-content/plugins/wpshopgermany-handlerbund kopieren, oder über das Wordpress Backend: Plugins > Installieren > Hochladen die ZIP Datei hochladen.
2. Das Plugin im Plugin Menü von Wordpress aktivieren.
3. Seiten für die Texte anlegen. (Im Wordpress Menü "Seiten")
4. Die Accountdaten vom Händlerbund in die Einstellungen unter Einstellungen -> Händlerbund eintragen.
5. Die Seiten auswählen auf denen die Texte vom Händlerbund eingesetzt werden sollen, oder neue anlegen.
5. Haken bei "Abgleichen" setzen und den Button "Speichern" betätigen.

Details auch in der [Dokumentation](http://wpshopgermany.maennchen1.de/?p=3426 "Dokumentation des wpShopGermany Wordpress Shop Plugin Moduls wpshopgermany-handlerbund")
